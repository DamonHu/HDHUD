## 特点

简单易用，通过不同的参数，实现不同的展示效果

* 纯文本展示
* 带icon展示，icon和图片的位置支持自定义
* 进度条展示
* loading展示
* 自适应文本大小


|功能|函数示例|
|----|----|
|纯文本展示|`HDHUD.show("纯文本展示")`|
|icon横版排列|`HDHUD.show("横版排列",hudType: .warn, direction: .horizontal)`|
